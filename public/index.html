数据库表结构

users表

CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    username TEXT,    -- 用户名
    password TEXT,    -- 密码
    email TEXT,      -- 邮箱
    is_admin INTEGER -- 是否管理员(1是，0否)
);

products表

CREATE TABLE products (
    id INTEGER PRIMARY KEY,
    name TEXT,       -- 产品名称
    price REAL,      -- 价格
    description TEXT -- 描述
);

示例数据：

  • users表包含管理员(admin)和普通用户(user1, user2)的信息
  • products表包含多个产品信息(Product A, B, C)
